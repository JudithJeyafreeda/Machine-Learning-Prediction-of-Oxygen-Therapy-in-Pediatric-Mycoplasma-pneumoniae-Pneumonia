This repository contains the python code used to train the machine learning models presented in the "Machine Learning Prediction of Oxygen Therapy in Pediatric Mycoplasma pneumoniae Pneumonia" paper.
The file train_part1.py is the code to train a "Logistic Regression", "Random Forest", "XGBoost", "SVM", "KNN", "Naive Bayes" and "Multi-layer Perceptron" models
The file train_part2.py is the code to train a "LightGBM" and "TabTransformer" model

The data used in the paper is private and thus cannot be shared. However, the meta data used can be found in the csv file
